# 📅 Habit Tracker

A clean, fully responsive habit tracking web app built with pure HTML, CSS and JavaScript. No frameworks, no backend, no account needed.

## ✨ Features

- **Tracker** — Monthly habit grid, check off days, weekly & daily progress
- **Stats** — Charts, consistency score, smart insights, streak board
- **Goals** — Weekly / monthly / yearly goals with progress charts
- **Heatmap** — Full-width activity heatmap for the whole year
- **Focus Mode** — Daily view with motivational quotes and 7-day timeline
- **Mobile Ready** — Fully responsive on all screen sizes with a dedicated mobile layout
- **CSV Export** — Download your monthly data
- **Local Storage** — All data saved in the browser automatically

## 🛠️ Tech Stack

- **HTML / CSS / JavaScript** — No frameworks
- **Chart.js** — For charts and graphs
- **Fonts** — IBM Plex Mono + Cairo (Google Fonts)

## 🚀 Getting Started

No installation needed. Just open the file:

```
index.html
```

Or clone and open:

```bash
git clone https://github.com/v4nixxpy/habit-tracker.git
cd habit-tracker
# open index.html in your browser
```
